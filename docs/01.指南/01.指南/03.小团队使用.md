## XrayR

- **1.首先配置DNS文件**
``` html 
vi /etc/XrayR/dns.json
``` 

  全部删除覆盖

``` html 
{
    "servers": [
        "1.1.1.1",
        "8.8.8.8",
        "localhost",
  {
        "address": "4.4.4.4（以实际为准）", 
        "port": 53,
        "domains": [
           "geosite:netflix" ,"geosite:disney"
        ]
      }
    ],
    "tag": "dns_inbound"
}
``` 

- **2.修改XrayR配置文件**

第5行去掉#号开启
   
``` html 
  DnsConfigPath:  /etc/XrayR/dns.json
```

第35行附近修改参数

``` html 
 EnableDNS: true 
 DNSType: UseIP
```

- **3.重启**
XrayR restart
