## XrayR

- **1.首先配置 DNS 文件**

```shell
vi /etc/XrayR/dns.json
```

全部删除覆盖

```json
{
  "servers": [
    "1.1.1.1",
    "8.8.8.8",
    "localhost",
    {
      "address": "4.4.4.4（以实际为准）",
      "port": 53,
      "domains": ["geosite:netflix", "geosite:disney"]
    }
  ],
  "tag": "dns_inbound"
}
```

- **2.修改 XrayR 配置文件**

第 5 行去掉#号开启

```json
  DnsConfigPath:  /etc/XrayR/dns.json
```

第 35 行附近修改参数

```json
 EnableDNS: true
 DNSType: UseIP
```

- **3.重启**

```shell
XrayR restart
```
