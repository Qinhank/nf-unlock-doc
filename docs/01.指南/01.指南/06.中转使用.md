# 进阶说明

本站中转途径分为两种，第一：直接国内入口连到你的落地，国内入口--你的落地；第二：使用特定落地机已经搭建好的隧道，国内入口--指定隧道--你的落地

使用国内机，默认需要协议自在TLS，不带TLS的协议请使用隧道中转，否则你的落地有被墙的风险。

使用隧道机时，可转发任意协议流量，可以使用对应地区的AZ实现免流，可以确保你的IP和端口安全，但因为隧道落地端有被墙和被风控的可能，所以有中断换IP等待恢复的风险。
