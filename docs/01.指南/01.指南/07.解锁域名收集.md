## 解锁域名收集

方便将需要解锁的内容，添加到DNS的分流设置规则内
示例添加奈飞，迪士尼和动画疯分流：
``` json
   "dns": {
    "servers": [
      "1.1.1.1","8.8.8.8", 
      {
        "address": "4.4.4.4（以实际为准）", 
        "port": 53,
        "domains": [
           "geosite:netflix","geosite:disney",gamer.com.tw,bahamut.com.tw,hinet.net,fbcdn.net,gvt1.com,digicert.com,viblast.com
        ]
      }
    ]
  }
```
